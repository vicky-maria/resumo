Como deixar um texto em negrito?

Existem duas formas de deixar um texto em negrito. Tem a forma semântica e a não semântica

Vc pode usar a tag <b> para deixar o texto em negrito, porém, ela não possui significado e só vai mudar a forma do texto.

Vc também pode usar a tag <strong> para deixar o texto em negrito. Essa tag além de mudar a forma do texto também indica que ele tem força.

Como deixar o texto em itálico?

Existem duas formas de deixar um texto em itálico. Existe a forma semântica e a não semântica.

Vc pode usar a tag <i> para deixar o texto em itálico, porém, ele só vai mudar a forma do texto e não tem significado.

Vc também pode usar a tag <em> para deixar o texto em itálico. Essa tag além de mudar a forma do texto também dá ênfase a esse texto.

Conclusão

O ideal é usar apenas as tags semânticas.

Como colocar um conteúdo dentro de uma tag?

Selecione o conteúdo, depois aperte ctrl+shift+p e depois escreva qual a tag onde vc quer colocar o conteúdo e aperte enter.

Marcar um trecho do texto

Sabe quando vc está estudando através de um livro e vc usa um marcador de texto pra sublinhar algo importante? É possível fazer algo parecido no HTML. Usando a tag <mark> vc pode sublinhar um trecho como se estivesse usando um marcador de texto no documento HTML. Para mudar a cor do sublinhado use CSS. EX: A vida é bela.

Deletado e inserido

A tag <del> indica que o trecho do texto pode ser lido, mas deve ser desconsiderado. EX: Eu não estou bem.

Já a tag <ins> indica que aquele trecho foi inserido e que deve prestar atenção nele. Ex: Eu não estou bem.

Subscrito e sobrescrito

A tag <sup> deixa a letra ou o que quer que seja pequininha e bem lá no alto. Ex: x30, 405

Já a tag <sub> deixa a letra ou o que quer que seja bem pequenininho e lá embaixo. Ex: H2O

Citações

Vc pode colocar citações no seu documento de duas formas: Use a tag <q> para adicionar citações simples, ou seja, citações mais curtas. ex: Uma vez a minha avó disse: água mole, pedra dura, tanto bate até que fura.

Sim, eu poderia colocar apenas as aspas para indicar que é uma citação, porém, essa tag tem significado e por isso é recomendado usá-la ao invés de só colocar aspas.

Use a tag <blockquote> para adicionar citações de blocos, ou seja, citações longas que tenham quebra de linha. ex: Arky disse para Mercedes:

Sad experiences are sad no matter how many times you've experienced them... Please don't just ignore it because you're used to it.
Use o atributo cite para colocar um link de um site relacionado a fonte de onde vc tirou a citação, como o site do autor ou o livro de onde tirou.

Abreviações

Para indicar abreviações, use a tag <abbr title='significado da abreviação'>. O significado que está no title vai ser exibido em um pequeno pop-up ao passar o mouse por cima da abreviação.

Código fonte

Para adicionar códigos ao seus documento use a tag <code>. Ela vai deixar a fonte monoespaçada, facilitando a leitura do código. A fonte monoespaçada deixa as letras com larguras iguais. Veja a diferença:
document.getElementById('teste')
document.getElementById('teste')

Para códigos curtos o <code> sozinho é suficiente. Mas e quando o código é maior?
num = int(input('Digite um número')) if num % 2 == 0: print(f'o número {num} é ímpar') print('fim do programa') Percebeu? O código não dica indentado. Como vimos anteriormente os espaços adicionais não são exibidos quando a página é carregada e por isso o código fica todo junto. Para exibir as identações precisamos colocar todo esse código dentro da tag <pre>. 


                num = int(input('Digite um número'))
                    if num % 2 == 0:
                    print(f'o número {num} é ímpar')
                    print('fim do programa')

Agora sim o código ficou bonito. Se vc quiser vc pode mudar a tabulação do código. Selecione o código e aperte shift + tab pra tirar espaços ou aperte tab para adicionar espaços.
